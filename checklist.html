Project Requirements

  • ☐ GitHub URL
  • ☐ Project Report, Upload zipped folders all website files with report
  • ☑ Minimum of 4 HTML Page Website
  • ☑ HTML5 standard elements and recommended layout structure
  • ☑ Include Meta data - description, keywords, title
  • ☑ Include font family for body of page
  • ☑ Include navigation and indicate the current page
  • ☑ Appropriate use of external style sheet
  • ☑ Appropriate example of internal styling
  • ☑ Appropriate example of inline styling
  • ☑ Home page content neat, aligned, visually attractive
  • ☑ Consistency of design on all pages, aligned, use of fonts, colour
  • ☑ Appropriate use of headings (h1, h2, h3, h4)
  • ☑ Include an article, section, and aside sections
  • ☑ Appropriate use of colours
  • ☑ Clear, consistent and align content on page(sources)
  • ☑ Include public domain images (at least one floated)
  • ☑ Use of background image
  • ☑ Include table of data/statistical information only
  • ☑ Include both ordered and unordered lists
  • ☑ Include favicon and logo
  • ☑ Include a contact form and display values entered
  • ☑ Verify data collected on contact form using GET method
  • ☑ Include Google Maps
  • ☑ Include newspaper columns using CSS column property
  • ☑ Footer information include copyright and footer navigation menu
  • ☑ Internal and external links
  • ☑ Pages should be up to 70% responsive
  • ☑ Include suitable video
  • ☑ Appropriate use of CSS Id and Class selectors
  • ☑ Appropriate use content layout using margins
  • ☑ Appropriate use content layout using paddings and borders
  • ☑ Additional Features
  • ☐ W3C HTML & CSS Validator results(in report)
  • ☐ Report include proposal, declaration, URL to GitHub, sketch, descriptions, sources and validator results
  • ☑ Website appearance, design, and content
